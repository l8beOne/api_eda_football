## История отчистки
1. после мерджа было ~21к строк
2. удаленны строки с пустым рейтингом (у таких игроков большенство полей так же были пустыми) ~14к строк
3. удаленны полные и неявные дубликаты ~9.7к строк


## Заполнение пропусков
* Вес был заполнен на основе нормалльного распределения для людей с таким же ростом
* У игроков, с отсутствующим ростом и весом они были заполненны по глабальному распределению
* Желто-красные карточки были заполненны на основе желтых и красных
* Число голов было заполнеено нулем у всех вратарей кроме Игорька
